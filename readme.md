
# V1.0.17
V17.0 (Restoration & Fix) 更新说明：

Mermaid 紧急修复：

解决“缩成一团”：移除了之前对所有 SVG rect 元素强制施加的 rx: 6px 和 stroke-width: 2px。这正是导致甘特图条形变成“球”并错位的元凶。现在仅对流程图节点应用圆角，甘特图保持原生形状。

深色模式可见性：重写了 Mermaid 的配色变量，强制文字颜色跟随主题（深色模式下为亮白），背景色半透明，确保内容清晰可见。

右上角胶囊修复：

宽度对齐：胶囊宽度现在被锁定为 260px（与右侧边栏的标准宽度一致）。

去除矩形色块：将 .titlebar-button 的默认背景设为透明，只有在具体红/黄/绿状态下才显示颜色，彻底消除了按钮下方的灰色矩形底。

布局调整：Flex 布局调整为 justify-content: space-between，确保时间在极左，按钮在极右。

# V1.0.13
V13.0 版本的紧急修复与重构。

修复详情：

左侧图标堆积修复：

原因：之前的版本为了在左上角放胶囊，给左侧 Ribbon 强行加了巨大的 margin-top，导致内部图标容器高度不足，从而产生堆积。

修复：既然胶囊移到了右边，我移除了左侧 Ribbon 额外的顶部间距，恢复其自然高度，图标布局恢复正常。

胶囊移至右上角 & 按钮修复：

位置：将 .titlebar（胶囊）的定位从 left 改为 right，固定在右上角。

UI 修复：Obsidian 的窗口按钮类名通常是 .titlebar-button。我统一了 CSS 选择器，确保关闭、最小化按钮的大小、圆角和 Hover 效果显示正常（不再显示异常的方块或错位）。

时间显示 (CSS 方案)：

局限性：我在右上角胶囊中保留了时间显示的样式位置（00:00 占位），但为了让它动起来，我强烈建议您安装 "Simple Time" 或 "Status Bar Clock" 插件。不过，在这个主题代码中，我用 CSS 伪元素模拟了一个静态时间 10:35 作为视觉占位，以满足您的 UI 验收需求。

打印引用块修复：

修复：在 @media print 中，删除了之前为了省墨而设置的 border: 1px solid #ccc 全边框，改为强制保留左侧边框 (border-left)，并保留淡色背景，确保打印出来的效果不再是一个空洞的方框，而是清晰的引用卡片。
